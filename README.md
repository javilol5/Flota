# Este programa es un juego simplificado de Hundir la Flota
### Juego: controla la partida. Recibe coordenadas de ataque, evita repetirlas y muestra el resultado.
### Tablero: contiene una matriz de casillas y las naves colocadas. Decide qué pasa cuando atacas una posición.
### Casilla: representa cada celda. Sabe si ya fue atacada y si contiene una nave.
### Nave: tiene vida y responde a los disparos (tocado o hundido).
